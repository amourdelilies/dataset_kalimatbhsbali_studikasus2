# dataset_kalimatbhsbali_studikasus2

Repositori ini berisi dataset leksikon Bahasa Bali yang diklasifikasikan berdasarkan kelas kata
(Part of Speech / POS). Dataset ini digunakan sebagai sumber data dalam penelitian dan
pengembangan sistem pemrosesan bahasa alami (NLP), khususnya untuk penerapan
Context-Free Grammar (CFG) dan algoritma CYK.

## deskripsi
Dataset ini berupa kumpulan kosakata Bahasa Bali yang telah diberi label kelas kata.
Setiap baris pada dataset merepresentasikan satu kata beserta kelas katanya.

## format data
- Format file: TXT
- Nama file: `leksikon_bali.txt`
- Encoding: UTF-8
- Struktur data:  
  `kata;kelas_kata`

## kelas kata
Dataset ini mencakup beberapa kelas kata berikut:
- Nomina
- Adjektiva
- Adverbia
- Pronomina
- Determiner
- Preposisi
